# Flutter WebView Starter Template

A production-ready Flutter WebView app template with:
- 2 splash screens
- Full WebView integration
- Offline detection
- Android back button support

## Placeholders
Before building, these are replaced automatically:
- `__APP_NAME__` - App display name
- `__PACKAGE_NAME__` - Android package (e.g. com.example.app)
- `__IOS_BUNDLE_ID__` - iOS bundle ID
- `__BASE_URL__` - Primary web URL
- `__ALLOWED_DOMAINS__` - Comma separated allowed domains
- `__APP_NAME_SNAKE__` - Snake case app name for pubspec

## Build
```bash
flutter pub get
flutter build apk --release
```
